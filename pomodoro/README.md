# Aesthetic Pomodoro Productivity App

A beautifully calm Pomodoro productivity timer built with Vanilla HTML, CSS, and JS. 

## Features
- **3 Modes**: Focus (25 min), Short Break (5 min), Long Break (15 min).
- **Auto-transition**: Automatically switches to the correct short or long break after completing a session.
- **Glassmorphism UI**: Uses a premium, clean glass-like effect to display the timer.
- **Calm Transitions**: The background features a smooth, rotating calm gradient that shifts colors based on the current mode.
- **Micro-Animations**: An animated circular progress ring smoothly ticks down along with the timer.

## How to Run
It's a pure static front-end application. No backend or build steps are required.

1. Open `index.html` directly in any modern web browser.
   - Or start a quick local server using: `python -m http.server 8000`
2. Navigate to `http://localhost:8000/pomodoro/index.html` (if using the local server).

## Technologies
- **HTML5** 
- **Vanilla CSS3** (Custom properties, Flexbox, Animations, Glassmorphism)
- **Vanilla JS** (Timer logic, DOM manipulation)
